┌──────────────────────────────────────────────────────────────────────────────┐
│ HOW IT WORKS                                                                 │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ 1. Device powers on and initializes Bluetooth stack                         │
│ 2. Custom GATT service registered with two characteristics                  │
│ 3. BLE advertising starts with custom service UUID                          │
│ 4. Central device (phone) scans and discovers "Battery Monitor"             │
│ 5. Central connects and performs service discovery                          │
│ 6. Central enables notifications on voltage characteristic                  │
│ 7. Battery monitor thread wakes every interval (default 1s)                 │
│ 8. Thread simulates voltage reading and updates characteristic              │
│ 9. Notification sent to central with new voltage value                      │
│ 10. Central can write new interval to change sampling rate                  │
│ 11. Process repeats with new interval                                       │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
